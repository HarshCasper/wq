# useContext(Context)

Standard react hook.

> Note that in wq, it is generally not necessary to useContext() directly.  Instead, the following hooks may be more useful:
> * [useModel]
> * [useList]
> * [usePluginReducer]
> * [useRenderContext]

```js
const contextVal = useContext(Context);
```
